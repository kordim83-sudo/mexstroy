Как настроить:
1. Создайте проект в Google Cloud Console
2. Включите Google Sheets API
3. Создайте OAuth 2.0 Client ID (тип: Web) и API Key
4. В настройках OAuth добавьте ${location.origin} в Authorized Origins
5. Вставьте ключи выше и нажмите «Подключиться»
